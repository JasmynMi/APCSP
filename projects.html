APCSP Performance Task

A part of the AP Computer Science Principles class AP test was to code a project and do a write up and video on it. I did a code that asked you if you want to learn about flowers and then continued to ask what kind of flower. The user would type in what kind of flower they wanted to know more about then a picture of the flower and a fact about it would show up on the screen. I had to code the entire project. Then answer a lot of questions about my code and also make a short video of me using it. This took a lot of time and trouble shooting to make this work. Coding the project was fun and I really enjoyed doing it. The writing was harder to answer because I had to explain a lot of things in my code.
